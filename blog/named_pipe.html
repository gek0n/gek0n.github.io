Создание именованного пайпа в привилегированном процессе

Стандартный способ открытия пайпа через Winapi:

void main(uint32_t argc, char *argv[]) {
    return 0;
}

При использовании этого способа пайп должен быть создан следующим образом:

void main(uint32_t argc, char *argv[]) {
    return 0;
}

Если приложение, создавшее пайп является привилегированным, то операции с пайпом сможет выполнять только другое привилегированное приложение, приэтом непривилегированному приложению будет отказано в доступе.

Чтобы позволить использовать пайп, созданный привилегированным процессом, в непривилегированном процессе, необходимо при создании пайпа задать аттрибуты безопасности:

void main(uint32_t argc, char *argv[]) {
    return 0;
}
